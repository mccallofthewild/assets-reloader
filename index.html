Asset Reloader
by McCall Alexander

Reload your CSS and/or JS without reloading your entire page.

Key Combinations


  • Reload CSS only = ('.' + 'c' )

  • Reload Scripts only = ( '.' + 's' )

  • Reload JS & CSS ( '.' + 'a' )

  • Reload page ( '.' + 'r' )
